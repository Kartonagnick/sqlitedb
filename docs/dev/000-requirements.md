Требования  
----------

- [x] Весь код библиотеки должен быть расположен в пространстве имен `db`  
- [x] В хэдэрах не должна отсвечивать зависимость от sqlite3.  
- [x] Thread-safe.  
- [x] `db::connection` не может иметь "дефолтно-пустое" состояние.  
  - [x] Cоздать объект можно лишь при помощи `db::connect`  
  - [x] `db::connect` сразу же подключает соединение к базе.  
  - [x] Такая мера необходима, что бы:  
    - [x] Избавиться от проверки: валидно ли соединение.  
    - [x] Избавиться от необходимости использовать мутекс.  
- [x] Деструкторы защищены assert.  
- [x] В релизе деструкторы игнорируют проблемы.
